# Cipher-Trails
Cipher Trails — 2D платформер-лабиринт на C# / MonoGame с ограниченной видимостью (туман войны). Финальный проект по программированию, 1 курс.

**Cipher Trails** — это 2D игра в жанре «лабиринт + платформер».  
Игрок исследует запутанные уровни, прыгает по платформам и ищет выход, полагаясь только на небольшой участок видимой карты.  
Проект создан в качестве **финальной работы** по дисциплине «Программирование» на 1 курсе университета.

---

## 🎮 Игровой процесс

- Управляй персонажем, двигаясь по **лабиринту из платформ и стен**.
- Твоя цель — добраться до **выхода** (дверь / портал).
- **Уникальная фишка**: видимость ограничена небольшим радиусом вокруг героя («туман войны»). Приходится запоминать путь или полагаться на интуицию.
- Преодолевай препятствия (шипы, враги — будут добавлены позже).

## 🕹️ Управление

| Действие       | Клавиша           |
|----------------|-------------------|
| Движение влево | `A` или `←`       |
| Движение вправо| `D` или `→`       |
| Прыжок         | `Space`           |
| Выход из игры  | `Esc`             |

## 🚀 Запуск проекта

### Требования
- [.NET SDK](https://dotnet.microsoft.com/download) (6.0 или выше)
- [MonoGame](https://www.monogame.net/) (устанавливается через шаблоны)

### Инструкция

```bash
# Клонируй репозиторий
git clone https://github.com/твой-логин/CipherTrails.git

# Перейди в папку проекта
cd CipherTrails

# Восстанови зависимости
dotnet restore

# Собери и запусти
dotnet run
